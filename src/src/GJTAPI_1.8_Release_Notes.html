GJTAPI 1.8 Release Notes

Version

This is version 1.8 of the Generic JTAPI, Jcat and Jcc project.

New Features

  • SIP Provider
  • TAPI3 Provider
  • XTAPI Bridge now moved into same licence and the core jar file.

Known Bugs

  • The new TAPI 3.0 Provider does not yet support playing or recording sounds

Install

  1. Add the JTAPI, Jcat or Jcc interface jar files on your classpath
  2. Add the GJTAPI-1.8 jar file onto your classpath
  3. Add any DLL or SO file required by the provider onto your library path
  4. Add any dependent jar files for your provider onto your classpath
  5. Create your Provider with the "Provider String" given in the following chart.
