GJTAPI 1.9 Release Notes

Version

This is version 1.9 of the Generic JTAPI, Jcat and Jcc project.

New Features

  • New provider lookup based on .gjtapi file extension

  • Providers now split into separate zip files

  • Required jar files included in provider zip files when platform neutral

  • New providers for JavaSound, MjSip and NjIax

Known Bugs

  • The TAPI 3.0 Provider does not yet support playing or recording sounds

Install

  1. Unzip the Gjtapi-core-1.9.zip file and any required provider zip files into the same directory

  2. Add redistributed jar files to the classpath

  3. Add the GJTAPI-1.9 jar file onto your classpath

  4. Add any DLL or SO file required by the provider onto your library path

  5. Create your Provider with the "Provider String" given in the following chart.
